Overview

HeartbeatManager is a module that is meant to be included in a manager class. The HeartbeatManager supports dynamic adding and removal of members and a custom heartbeat interval. The HeartbeatManager broadcasts to its ‘members’ initially with a heartbeat interval and a payload that indicates how the clients should respond to the heartbeat interval.
